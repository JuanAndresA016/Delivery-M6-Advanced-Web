# Delivery-M6-Advanced-Web

Instrucciones para ejecutar el backend:

Desde la carpeta "WebDelivery" abrir la ruta del backend y correr la aplicación "demo"

Instrucciones para ejecutar el frontend

Desde la carpeta "WebDelivery" ejecutar el comando "npm run dev" despues de haber instalado las dependencias con npm install


Descripción breve del proyecto:

El proyecto consiste en un formulario que te permita construir la armadura de un personaje de minecraft con opciones pre-establecidas, puedes personalizar y combinar cualquier partes de armadura de diamante, oro, netherite y hierro.
